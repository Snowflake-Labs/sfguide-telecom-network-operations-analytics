# Telecom Network Operations Analytics with Snowflake Intelligence

## Overview

Build an end-to-end telecom network operations reporting and analytics solution using Snowflake Intelligence. This guide demonstrates how to deploy a 4-persona Streamlit dashboard for Network Engineers, Performance Analysts, Network Managers, and Executives, along with a Snowflake Intelligence agent for natural language queries across RAN, Core, and Transport network data.

## Step-By-Step Guide

For prerequisites, environment setup, step-by-step guide and instructions, please refer to the [Snowflake Guide](https://www.snowflake.com/en/developers/guides/telecom-network-ops-intelligence/).
